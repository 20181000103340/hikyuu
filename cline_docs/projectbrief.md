# Project Brief

## Project Name
Hikyuu Quant Framework

## Core Mission
提供基于C++和Python的量化交易研究框架，支持回测、实盘交易演练等功能。

## Objectives
1. 高性能的C++核心引擎
2. 友好的Python接口
3. 完整的量化交易工具链
4. 灵活的策略开发支持
5. 可靠的数据管理系统

## Technical Constraints
- C++/Python混合开发
- 跨平台支持
- 高性能要求
- 数据一致性保证

## Project Scope
- 量化交易核心功能
- 数据获取与管理
- 策略开发与回测
- 实盘交易接口
- 可视化分析工具
